== Description ==
A free business directory plugin with premium features included. Requires the Advanced Custom Fields plugin to be truly useful.

[Demo](https://hiresamwright.com/servers/hsw-directory/ "WordPress Business Directory")

* List any kind of businesses.
* Mapbox powered Mapping with Geo Location address finder.
* Easy template overrides.
* Customise the colors and branding.
* Auto map with paging.

Add the shortcode [hsw_directory] and go to Appearance > Customise > HSW Directory to configure the options.

Mapbox
Maps are powered by Mapbox you will need an Access Token to use the map feature.

Override Template
Copy the contents of ‘/hsw-directory/templates/’ to ‘/your-child-theme/hsw-directory/templates/’ and edit the files to your liking.

Set the Filters by URL
Use a URL in the format /?taxonomy_1=[term_id]&taxonomy_2=[term_id]

== Installation ==
Download from Admin >  Plugins > Add New, use the Customizer to override the defaults.
